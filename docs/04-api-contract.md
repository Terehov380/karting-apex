# API-контракт клиентского приложения

## 1. Назначение

Документ описывает HTTP API, предоставляемый существующим backend картинг-центра «Апекс» для клиентского веб-приложения. Backend является black-box источником истины; данный контракт — единственная каноническая схема данных.

## 2. Общие сведения

| Параметр | Значение |
|----------|----------|
| **Base URL** | `https://api.karting-apex.ru/v1` (продакшн) |
| **Base URL (mock)** | Используется `mock-api.js` в учебной версии |
| **Формат** | JSON (все запросы и ответы) |
| **Авторизация** | TBD — механизм идентификации клиента уточняется у заказчика. Условный пример: `Authorization: Bearer <token>` |
| **Кодировка** | UTF-8 |
| **Content-Type** | `application/json` |

## 3. Общие ошибки

### 3.1. Формат ошибки

```json
{
  "status": 400,
  "code": "VALIDATION_ERROR",
  "message": "Описание ошибки на русском языке",
  "details": [
    { "field": "participants", "message": "Количество участников превышает свободные карты" }
  ]
}
```

### 3.2. HTTP-статусы ошибок

| Статус | Код | Описание |
|--------|-----|----------|
| **400** | `BAD_REQUEST` | Некорректный запрос (неверный JSON, отсутствуют обязательные поля) |
| **401** | `UNAUTHORIZED` | Отсутствует или недействительна учётная запись клиента (способ идентификации — TBD) |
| **404** | `NOT_FOUND` | Ресурс не найден (слот, бронирование) |
| **409** | `CONFLICT` | Конфликт: слот уже занят, двойная бронь, отмена невозможна |
| **422** | `UNPROCESSABLE_ENTITY` | Валидация полей запроса не пройдена |
| **500** | `INTERNAL_SERVER_ERROR` | Внутренняя ошибка сервера |
| **503** | `SERVICE_UNAVAILABLE` | Сервис временно недоступен |

## 4. Endpoints: Слоты

### 4.1. GET /slots

Получить список доступных слотов.

**Запрос:**

| Параметр | Тип | Обязательный | По умолчанию | Описание |
|----------|-----|:------------:|:------------:|----------|
| `dateFrom` | `string` (date) | нет | текущая дата | Начало периода (YYYY-MM-DD) |
| `dateTo` | `string` (date) | нет | dateFrom + 7 дней | Конец периода (YYYY-MM-DD) |

**Успешный ответ: 200**

```json
{
  "slots": [
    {
      "id": "slot-001",
      "startTime": "2026-07-05T10:00:00Z",
      "endTime": "2026-07-05T10:20:00Z",
      "availableKarts": 8,
      "maxParticipants": 8,
      "pricePerPerson": 1000,
      "status": "available",
      "track": {
        "id": "track-short",
        "name": "Короткая трасса",
        "description": "Трасса для новичков, 600 м"
      },
      "marshal": {
        "id": "mar-001",
        "name": "Алексей С."
      },
      "location": {
        "address": "ул. Картинговая, 1, окраина города",
        "meetingPoint": "Зона регистрации у входа"
      }
    }
  ]
}
```

**Ошибки:** 400, 401, 500, 503

### 4.2. GET /slots/{slotId}

Получить детальную информацию о конкретном слоте.

**Запрос:**

| Параметр | Тип | Обязательный | Описание |
|----------|-----|:------------:|----------|
| `slotId` | `string` | да | Идентификатор слота (path) |

**Успешный ответ: 200** — объект `Slot` (см. п. 4.1)

**Ошибки:** 400, 401, 404, 500, 503

## 5. Endpoints: Бронирования

### 5.1. POST /bookings

Создать новое бронирование.

**Запрос:**

```json
{
  "slotId": "slot-001",
  "participants": 2,
  "equipment": [
    { "id": "helmet", "quantity": 2 }
  ],
  "customerName": "Иван Петров",
  "customerPhone": "+7 999 123-45-67"
}
```

| Поле | Тип | Обязательный | Описание |
|------|-----|:------------:|----------|
| `slotId` | `string` | да | Идентификатор выбранного слота |
| `participants` | `number` | да | Количество участников (1–maxParticipants слота) |
| `equipment` | `EquipmentRequest[]` | нет | Запрошенная экипировка (если не указана — клиент со своей) |
| `customerName` | `string` | да | Имя клиента |
| `customerPhone` | `string` | да | Телефон клиента |

**EquipmentRequest:**

| Поле | Тип | Обязательный | Описание |
|------|-----|:------------:|----------|
| `id` | `string` | да | Идентификатор позиции экипировки |
| `quantity` | `number` | да | Количество (≥ 1, ≤ participants) |

**Успешный ответ: 201**

```json
{
  "id": "bk-42",
  "slotId": "slot-001",
  "customerName": "Иван Петров",
  "customerPhone": "+7 999 123-45-67",
  "participants": 2,
  "equipment": [
    { "id": "helmet", "name": "Шлем", "pricePerUnit": 200, "quantity": 2 }
  ],
  "totalPrice": 2400,
  "status": "confirmed",
  "cancellationDeadline": "2026-07-05T09:50:00Z",
  "canCancel": true,
  "createdAt": "2026-07-04T12:00:00Z"
}
```

**Ошибки:**

| Статус | Условие |
|--------|---------|
| **400** | Некорректный JSON, отсутствуют обязательные поля |
| **401** | Неавторизован |
| **409** | Слот уже занят, свободных картов недостаточно; двойная бронь |
| **422** | Валидация полей не пройдена (participants > maxParticipants, неверный телефон и т.д.) |
| **500** | Внутренняя ошибка |
| **503** | Сервис недоступен |

**Пример ошибки 409:**

```json
{
  "status": 409,
  "code": "SLOT_FULL",
  "message": "Выбранный слот больше не имеет свободных картов. Пожалуйста, выберите другой слот."
}
```

### 5.2. GET /bookings

Получить список бронирований текущего клиента.

**Запрос:**

| Параметр | Тип | Обязательный | Описание |
|----------|-----|:------------:|----------|
| `status` | `string` | нет | Фильтр по статусу: confirmed / cancelled_by_client / cancelled_by_center / completed |

**Успешный ответ: 200**

```json
{
  "bookings": [
    {
      "id": "bk-42",
      "slotId": "slot-001",
      "startTime": "2026-07-05T10:00:00Z",
      "trackName": "Короткая трасса",
      "marshalName": "Алексей С.",
      "participants": 2,
      "totalPrice": 2400,
      "status": "confirmed",
      "cancellationDeadline": "2026-07-05T09:50:00Z",
      "canCancel": true,
      "createdAt": "2026-07-04T12:00:00Z"
    }
  ]
}
```

**Ошибки:** 401, 500, 503

### 5.3. GET /bookings/{bookingId}

Получить детальную информацию о конкретном бронировании.

**Запрос:**

| Параметр | Тип | Обязательный | Описание |
|----------|-----|:------------:|----------|
| `bookingId` | `string` | да | Идентификатор бронирования (path) |

**Успешный ответ: 200** — объект `Booking`

**Ошибки:** 400, 401, 404, 500, 503

### 5.4. POST /bookings/{bookingId}/cancel

Отменить бронирование.

**Запрос:**

```json
{
  "reason": "Передумали"
}
```

| Поле | Тип | Обязательный | Описание |
|------|-----|:------------:|----------|
| `reason` | `string` | нет | Причина отмены (опционально) |

**Успешный ответ: 200**

```json
{
  "id": "bk-42",
  "slotId": "slot-001",
  "status": "cancelled_by_client",
  "cancellationInfo": {
    "cancelledBy": "client",
    "reason": "Передумали",
    "cancelledAt": "2026-07-04T14:00:00Z"
  },
  "canCancel": false
}
```

**Ошибки:**

| Статус | Условие |
|--------|---------|
| **400** | Некорректный запрос |
| **401** | Неавторизован |
| **404** | Бронирование не найдено |
| **409** | Отмена невозможна (canCancel=false, просрочен cancellationDeadline, или бронь уже отменена/завершена) |
| **500** | Внутренняя ошибка |
| **503** | Сервис недоступен |

**Пример ошибки 409 при отмене:**

```json
{
  "status": 409,
  "code": "CANCELLATION_NOT_ALLOWED",
  "message": "Отмена данного бронирования невозможна: истёк срок отмены."
}
```

## 6. Типы данных (DTO)

### Slot

| Поле | Тип | Описание |
|------|-----|----------|
| `id` | `string` | Уникальный идентификатор слота |
| `startTime` | `string` (ISO 8601) | Дата и время старта |
| `endTime` | `string` (ISO 8601) | Дата и время окончания |
| `availableKarts` | `number` | Свободные карты на данный момент |
| `maxParticipants` | `number` | Максимум участников (обычно ≤ availableKarts) |
| `pricePerPerson` | `number` | Цена за одного участника (₽) |
| `status` | `string` | `available` / `full` / `cancelled` |
| `track` | `TrackConfiguration` | Конфигурация трассы |
| `marshal` | `MarshalSummary` | Маршал на слот |
| `location` | `LocationInfo` | Адрес и место сбора |

### TrackConfiguration

| Поле | Тип | Описание |
|------|-----|----------|
| `id` | `string` | Идентификатор конфигурации трассы |
| `name` | `string` | Название (например, «Короткая трасса», «Длинная трасса») |
| `description` | `string` | Описание трассы |

### MarshalSummary

| Поле | Тип | Описание |
|------|-----|----------|
| `id` | `string` | Идентификатор маршала |
| `name` | `string` | Имя маршала |

### LocationInfo

| Поле | Тип | Описание |
|------|-----|----------|
| `address` | `string` | Адрес картинг-центра |
| `meetingPoint` | `string` | Место сбора на территории центра |

### EquipmentOption

| Поле | Тип | Описание |
|------|-----|----------|
| `id` | `string` | Идентификатор (например, `helmet`, `balaclava`) |
| `name` | `string` | Название (например, «Шлем», «Подшлемник») |
| `pricePerUnit` | `number` | Цена за единицу (₽) |

### Booking

| Поле | Тип | Описание |
|------|-----|----------|
| `id` | `string` | Уникальный идентификатор бронирования |
| `slotId` | `string` | Ссылка на слот |
| `customerName` | `string` | Имя клиента |
| `customerPhone` | `string` | Телефон клиента |
| `participants` | `number` | Количество участников |
| `equipment` | `EquipmentBooking[]` | Заказанная экипировка |
| `totalPrice` | `number` | Итоговая стоимость (₽) |
| `status` | `string` | `confirmed` / `cancelled_by_client` / `cancelled_by_center` / `completed` |
| `cancellationDeadline` | `string` (ISO 8601) | Крайний срок отмены |
| `canCancel` | `boolean` | Разрешена ли отмена в текущий момент |
| `cancellationInfo` | `CancellationInfo \| null` | Информация об отмене (если есть) |
| `createdAt` | `string` (ISO 8601) | Дата и время создания |
| `startTime` | `string` (ISO 8601) | Дата и время старта заезда (для списка) |
| `trackName` | `string` | Название трассы (для списка) |
| `marshalName` | `string` | Имя маршала (для списка) |

### EquipmentBooking

| Поле | Тип | Описание |
|------|-----|----------|
| `id` | `string` | Идентификатор позиции |
| `name` | `string` | Название |
| `pricePerUnit` | `number` | Цена за единицу |
| `quantity` | `number` | Количество |

### CancellationInfo

| Поле | Тип | Описание |
|------|-----|----------|
| `cancelledBy` | `string` | `client` или `center` |
| `reason` | `string \| null` | Причина отмены |
| `cancelledAt` | `string` (ISO 8601) | Дата и время отмены |

### Notification

| Поле | Тип | Описание |
|------|-----|----------|
| `id` | `string` | Идентификатор уведомления |
| `type` | `string` | `booking_cancelled_by_center` / `booking_reminder` |
| `message` | `string` | Текст уведомления |
| `relatedBookingId` | `string` | Связанное бронирование |
| `read` | `boolean` | Прочитано ли уведомление |
| `createdAt` | `string` (ISO 8601) | Дата создания |

### APIError

| Поле | Тип | Описание |
|------|-----|----------|
| `status` | `number` | HTTP-статус |
| `code` | `string` | Код ошибки |
| `message` | `string` | Сообщение для пользователя |
| `details` | `FieldError[] \| null` | Детали по полям (опционально) |

### FieldError

| Поле | Тип | Описание |
|------|-----|----------|
| `field` | `string` | Имя поля |
| `message` | `string` | Описание ошибки |

## 7. Статусы бронирования

| Статус | Описание | Отображается клиенту |
|--------|----------|----------------------|
| `confirmed` | Бронь подтверждена | «Подтверждено» |
| `cancelled_by_client` | Отменено клиентом | «Отменено» |
| `cancelled_by_center` | Отменено центром (погода, тех. причины) | «Отменён центром» + причина |
| `completed` | Заезд проведён | «Завершено» |
